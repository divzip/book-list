## Installation
<pre>composer install</pre>
<pre>Create .env file with all correct values for DB</pre>
<pre>php artisan migrate:fresh --seed</pre>
<pre>npm install</pre>
<pre>npm run dev (or npm run prod)</pre>
<pre>php artisan serve</pre>
